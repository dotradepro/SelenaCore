Pair your Smart Life account with SelenaCore using your app authorization code — no developer account needed.
  1. Open the Smart Life (or Tuya Smart) app on your phone
  2. Tap Me in the bottom bar → ⚙️ icon top-right → Authorization code
  3. Copy the 6-8 character code (valid for ~10 minutes)
  4. Paste the code below and click "Start"
  5. SelenaCore will show a QR. Tap "+" in Smart Life → Scan → point at the QR
  6. Tap "Authorize" in the app. SelenaCore will automatically import all your devices.
