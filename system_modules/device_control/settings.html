  2. Tap Me in the bottom bar → ⚙️ icon top-right → Authorization code
  3. Copy the 6-8 character code (valid for ~10 minutes)
  4. Paste the code below and click "Start"
  5. SelenaCore will show a QR. Tap "+" in Smart Life → Scan → point at the QR
